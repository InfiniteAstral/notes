# 2.1 基本SQL

## 基本语法规则

- SQL 语句由**保留字**和用户自定义字组成，绝大多数组件**不区分大小写**
- 通常使用分号 `;` 作为**语句结束符**（SQL 标准未强制要求）
- SQL 为**自由格式**，通过缩进与换行可提升可读性：每个子句单独一行、子句开头对齐、子句多部分换行缩进

## 标识符与标量数据类型

### 标识符

用于标识**数据库对象**，如表名、视图名、列名等。

### 标量数据类型

| 数据类型   | 声明                                            |
| :--------- | :---------------------------------------------- |
| 布尔型     | `BOOLEAN`                                       |
| 字符型     | `CHAR`、`VARCHAR`                               |
| 位型       | `BIT`、`BIT VARYING`                            |
| 精确数值型 | `NUMERIC`、`DECIMAL`、`INTEGER`、`SMALLINT`     |
| 近似数值型 | `FLOAT`、`REAL`、`DOUBLE PRECISION`             |
| 日期时间型 | `DATE`、`TIME`、`TIMESTAMP`                     |
| 间隔型     | `INTERVAL`                                      |
| 大对象型   | `CHARACTER LARGE OBJECT`、`BINARY LARGE OBJECT` |

## 数据定义语言（DDL）

### 核心操作

1. **创建数据库**：ISO 标准**未统一规定**，各数据库方言实现方式不同
2. **创建表（`CREATE TABLE`）**

   可定义**列属性**（数据类型、`NOT NULL`、`UNIQUE`、`DEFAULT`、`CHECK`）与**表级约束**（主键、唯一键、外键、`CHECK`），支持**实体完整性、参照完整性、用户定义完整性约束**。

3. **修改表（`ALTER TABLE`）**

   支持**添加/删除列、添加/删除约束、设置/删除默认值**，可指定 `RESTRICT`（受限删除）、`CASCADE`（级联删除）。

4. **删除表（`DROP TABLE`）**

   语法：`DROP TABLE 表名 [RESTRICT|CASCADE]`

## 数据查询语言（DQL）

### 通用语法

```sql
SELECT [DISTINCT|ALL] {*|[列表达式 [AS 别名]][, ...]}
FROM 表名 [别名][, ...]
[WHERE 条件]
[GROUP BY 列列表]
[HAVING 条件]
[ORDER BY 列列表 [ASC|DESC]]
```

### 基础查询

::: info 例

1. 查询所有学生信息

   ```sql
   SELECT
     *
   FROM Student;
   ```

2. 查询选课表中不重复的学号

   ```sql
   SELECT DISTINCT
     sNo
   FROM SC;
   ```

3. 查询学生学号和姓名

   ```sql
   SELECT
     sNo,
     sName
   FROM Student;
   ```

4. 查询学生姓名及出生年份（2023 - 年龄）

   ```sql
   SELECT
     sName,
     2023 - sAge
   FROM Student;
   ```

:::

### 行选择（`WHERE` 子句）

支持**比较运算符**、范围（`BETWEEN AND`）、集合成员（`IN`）、模式匹配（`LIKE`）、空值判断（`IS NULL`）、逻辑运算符（`AND`、`OR`、`NOT`）。

- 通配符：`%` 匹配**任意长度字符**，`_` 匹配**单个字符**，`\` 为**转义符**

::: info 例

1. 查询所有男学生姓名

   ```sql
   SELECT
     sName
   FROM Student
   WHERE sex = '男';
   ```

2. 查询年龄 ≤17 岁的学生姓名

   ```sql
   SELECT
     sName
   FROM Student
   WHERE age <= 17;
   ```

3. 查询年龄不在 20~23 岁之间的学生信息

   ```sql
   SELECT
     sNo,
     sName,
     dNo,
     age
   FROM Student
   WHERE age NOT BETWEEN 20 AND 23;
   ```

4. 查询非 01/02/03 学院的学生姓名与性别

   ```sql
   SELECT
     sName,
     sex
   FROM Student
   WHERE dNo NOT IN ('01', '02', '03');
   ```

5. 查询张姓学生信息

   ```sql
   SELECT
     sNo,
     sName,
     sex
   FROM Student
   WHERE sName LIKE '张%';
   ```

6. 带转义符的课程名匹配查询

   ```sql
   SELECT
     *
   FROM Course
   WHERE cName LIKE 'DB\_%i__' ESCAPE '\';
   ```

7. 查询 030101 课程成绩为空的学号

   ```sql
   SELECT
     sNo
   FROM SC
   WHERE cNo = '030101'
     AND score IS NULL;
   ```

8. 复合逻辑条件查询

   ```sql
   SELECT
     *
   FROM Student
   WHERE dNo = '01'
     AND (age < 20 OR age > 23);
   ```

:::

### 结果排序（`ORDER BY`）

- 必须为 `SELECT` **最后一个子句**
- `ASC`：**升序**（默认）
- `DESC`：**降序**

::: info 例

1. 010101 课程按成绩降序查询学号与成绩

   ```sql
   SELECT
     sNo,
     score
   FROM SC
   WHERE cNo = '010101'
   ORDER BY score DESC;
   ```

2. 按学院升序、年龄降序查询所有学生

   ```sql
   SELECT
     *
   FROM Student
   ORDER BY dNo, age DESC;
   ```

:::

### 聚合函数

常用：`COUNT`、`AVG`、`SUM`、`MAX`、`MIN`；除 `COUNT(*)` 外，**均忽略空值**，`SUM`、`AVG` **仅适用于数值型**。

::: info 例

1. 统计学生总人数

   ```sql
   SELECT
     COUNT(*)
   FROM Student;
   ```

2. 统计学分 ≥2 的课程数（别名 `countOf`）

   ```sql
   SELECT
     COUNT(*) AS countOf
   FROM Course
   WHERE credit >= 2;
   ```

3. 统计 010101 课程平均成绩

   ```sql
   SELECT
     AVG(score)
   FROM SC
   WHERE cNo = '010101';
   ```

4. 统计选课的去重学生数

   ```sql
   SELECT
     COUNT(DISTINCT sNo)
   FROM SC;
   ```

5. 统计 001 学院课程总学分

   ```sql
   SELECT
     SUM(credit)
   FROM Course
   WHERE dNo = '001';
   ```

6. 查询 001 学院最小年龄

   ```sql
   SELECT
     MIN(age)
   FROM Student
   WHERE dNo = '001';
   ```

:::

### 分组查询（`GROUP BY`）

- `SELECT` 中**非聚合列必须出现在 `GROUP BY` 子句中**
- `WHERE` 子句**先过滤数据**，`GROUP BY` 子句**再分组**，`HAVING` 子句**过滤分组结果**
- 分组时两个 `NULL` **视为相等**

::: info 例

1. 按课程号统计选课人数

   ```sql
   SELECT
     cNo,
     COUNT(sNo)
   FROM SC
   GROUP BY cNo;
   ```

2. 筛选选课数 >3 且成绩 >60 的学生学号

   ```sql
   SELECT
     sNo
   FROM SC
   WHERE score > 60
   GROUP BY sNo
   HAVING COUNT(*) > 3;
   ```

3. 按学院统计学生数，筛选人数 >100 的学院

   ```sql
   SELECT
     dNo,
     COUNT(sNo)
   FROM Student
   GROUP BY dNo
   HAVING COUNT(sNo) > 100;
   ```

4. 按学院统计平均年龄

   ```sql
   SELECT
     AVG(age)
   FROM Student
   GROUP BY dNo;
   ```

5. 按课程统计最高/最低成绩

   ```sql
   SELECT
     MAX(score),
     MIN(score)
   FROM SC
   GROUP BY cNo;
   ```

6. 按学院统计课程总学分

   ```sql
   SELECT
     SUM(credit)
   FROM Course
   GROUP BY dNo;
   ```

:::

### 子查询

分为**标量子查询**（单行单列）、**行子查询**（单行多列）、**表子查询**（多行多列），支持**相关/非相关子查询**，可搭配 `ANY`、`SOME`、`ALL`。

::: info 例

1. 查询与赵敏同学院的学生姓名

   ```sql
   SELECT
     sName
   FROM Student
   WHERE dNo = (
     SELECT
       dNo
     FROM Student
     WHERE sName = '赵敏'
   );
   ```

2. 查询选修《矩阵论》的学生姓名

   ```sql
   SELECT
     sName
   FROM Student
   WHERE sNo IN (
     SELECT
       sNo
     FROM SC
     WHERE cNo = (
       SELECT
         cNo
       FROM Course
       WHERE cName = '矩阵论'
     )
   );
   ```

3. 查询王兵选修的所有课程名称

   ```sql
   SELECT
     cName
   FROM Course
   WHERE cNo IN (
     SELECT
       cNo
     FROM SC
     WHERE sNo = (
       SELECT
         sNo
       FROM Student
       WHERE sName = '王兵'
     )
   );
   ```

4. 处理 `NULL` 的子查询：查询无未成年学生的学院

   ```sql
   SELECT
     dName
   FROM Department
   WHERE dNo NOT IN (
     SELECT
       dNo
     FROM Student
     WHERE dNo IS NOT NULL
       AND age < 18
   );
   ```

5. 查询年龄小于平均年龄的学生

   ```sql
   SELECT
     sNo,
     sName
   FROM Student
   WHERE age < (
     SELECT
       AVG(age)
     FROM Student
   );
   ```

6. 年龄小于 01 学院任意学生的非 01 院学生

   ```sql
   SELECT
     sName,
     age
   FROM Student
   WHERE age < ANY (
     SELECT
       age
     FROM Student
     WHERE dNo = '01'
   )
     AND dNo <> '01'
   ORDER BY age DESC;
   ```

7. 年龄小于 01 学院所有学生的非 01 院学生

   ```sql
   SELECT
     sName,
     age
   FROM Student
   WHERE age < ALL (
     SELECT
       age
     FROM Student
     WHERE dNo = '01'
   )
     AND dNo <> '01'
   ORDER BY age DESC;
   ```

:::

### 多表查询

包括**内连接、自连接、外连接、多表连接、自然连接**，需合并多表列时必须使用连接。

::: info 例

1. 学生表与选课表内连接（全字段）

   ```sql
   SELECT
     Student.*,
     SC.*
   FROM Student, SC
   WHERE Student.sNo = SC.sNo;
   ```

2. 别名简化的内连接

   ```sql
   SELECT
     s.sNo,
     s.sName,
     sc.cNo,
     sc.score
   FROM Student s, SC sc
   WHERE s.sNo = sc.sNo;
   ```

3. 自连接：查询《数据库》的先修课名称

   ```sql
   SELECT
     c2.cName
   FROM Course c1, Course c2
   WHERE c1.cPNo = c2.cNo
     AND c1.cName = '数据库';
   ```

4. 自连接：查询先修课为《离散数学》的课程

   ```sql
   SELECT
     c1.cName
   FROM Course c1, Course c2
   WHERE c1.cPNo = c2.cNo
     AND c2.cName = '离散数学';
   ```

5. 左外连接：查询所有学生及选课信息（含未选课）

   ```sql
   SELECT
     s.*,
     sc.*
   FROM Student s
   LEFT OUTER JOIN SC sc
     ON s.sNo = sc.sNo;
   ```

6. 三表连接：学生-课程-成绩查询

   ```sql
   SELECT
     s.sName,
     c.cName,
     sc.score
   FROM Student s, Course c, SC sc
   WHERE s.sNo = sc.sNo
     AND c.cNo = sc.cNo;
   ```

7. 按学院筛选：查询软件学院学生的选课成绩

   ```sql
   SELECT
     s.sName,
     c.cName,
     sc.score
   FROM Student s, Department d, Course c, SC sc
   WHERE s.dNo = d.dNo
     AND s.sNo = sc.sNo
     AND c.cNo = sc.cNo
     AND d.dName = '软件学院';
   ```

:::

### `EXISTS`、`NOT EXISTS`

`EXISTS`、`NOT EXISTS` 表示**存在量词**，返回布尔值，子查询常用 `SELECT *`，可实现**全称量词转换**。

::: info 例

1. 查询未选修 01 号课程的学生姓名

   ```sql
   SELECT
     sName
   FROM Student
   WHERE NOT EXISTS (
     SELECT
       *
     FROM SC
     WHERE sNo = Student.sNo
       AND cNo = '01'
   );
   ```

2. 查询与赵一同学院的学生

   ```sql
   SELECT
     sNo,
     sName
   FROM Student S1
   WHERE EXISTS (
     SELECT
       *
     FROM Student S2
     WHERE S2.sDept = S1.sDept
       AND S2.sName = '赵一'
   );
   ```

3. 查询选修全部课程的学生姓名

   ```sql
   SELECT
     sName
   FROM Student s
   WHERE NOT EXISTS (
     SELECT
       *
     FROM Course c
     WHERE NOT EXISTS (
       SELECT
         *
       FROM SC
       WHERE sNo = s.sNo
         AND cNo = c.cNo
     )
   );
   ```

4. 查询选修 170101 同学全部课程的学生学号

   ```sql
   SELECT DISTINCT
     sNo
   FROM SC SCX
   WHERE NOT EXISTS (
     SELECT
       *
     FROM SC SCY
     WHERE SCY.sNo = '170101'
       AND NOT EXISTS (
         SELECT
           *
         FROM SC SCZ
         WHERE SCZ.sNo = SCX.sNo
           AND SCZ.cNo = SCY.cNo
       )
   );
   ```

:::

### 行内子查询

::: info 例
查询学生学号、姓名及平均成绩

```sql
SELECT
  sv1.sNo,
  s.sName,
  sv1.avg_score
FROM Student s,
     (
       SELECT
         sNo,
         AVG(score) AS avg_score
       FROM SC
       GROUP BY sNo
     ) AS sv1(sNo, avg_score)
WHERE s.sNo = sv1.sNo;
```

:::

### 集合运算

`UNION`（并）、`INTERSECT`（交）、`EXCEPT`（差），要求查询结果**列数与类型兼容**。

::: info 例

1. 并集：查询选修 030101 或 030102 课程的学号

   ```sql
   SELECT DISTINCT
     sNo
   FROM SC
   WHERE cNo = '030101'
   UNION
   SELECT
     sNo
   FROM SC
   WHERE cNo = '030102';
   ```

2. 交集：查询同时选修 030101和 030102 课程的学号

   ```sql
   SELECT DISTINCT
     sNo
   FROM SC
   WHERE cNo = '030101'
   INTERSECT
   SELECT
     sNo
   FROM SC
   WHERE cNo = '030102';
   ```

:::

## 数据操纵语言（DML）

### 插入数据（INSERT）

支持**单行插入**、**批量插入**（查询结果插入）。

::: info 例

1. 单行插入学生数据

   ```sql
   INSERT INTO Student
   VALUES ('170120', '陈冬', '男', 18, NULL, '01');
   ```

2. 创建成绩表并批量插入平均成绩数据

   ```sql
   CREATE TABLE s_score (
     sNo CHAR(6),
     sName VARCHAR(20),
     avg DECIMAL(5, 2)
   );

   INSERT INTO s_score (sNo, sName, avg)
   SELECT
     s.sNo,
     s.sName,
     v.avgscore
   FROM Student s,
        (
          SELECT
            sNo,
            AVG(score) AS avgscore
          FROM SC
          GROUP BY sNo
        ) AS v(sNo, avgscore)
   WHERE s.sNo = v.sNo;
   ```

3. 部分列插入选课数据

   ```sql
   INSERT INTO SC (sNo, cNo)
   VALUES ('170120', '010101');
   ```

:::

### 修改数据（UPDATE）

修改表中数据，支持**子查询作为条件**。

::: info 例

1. 修改指定学生的年龄

   ```sql
   UPDATE Student
   SET age = 22
   WHERE sNo = '170101';
   ```

2. 所有学生年龄加 1

   ```sql
   UPDATE Student
   SET age = age + 1;
   ```

3. 按姓名和课程修改成绩

   ```sql
   UPDATE SC
   SET score = 60
   WHERE sNo = (
     SELECT
       sNo
     FROM Student
     WHERE sName = '张敏'
   )
     AND cNo = (
       SELECT
         cNo
       FROM Course
       WHERE cName = '数据库系统'
     );
   ```

:::

### 删除数据（DELETE）

删除表中数据，支持**条件删除**，无 `WHERE` 则**删除全部数据**。

::: info 例

1. 删除指定学号的学生

   ```sql
   DELETE FROM Student
   WHERE sNo = '070122';
   ```

2. 清空学生表所有数据

   ```sql
   DELETE FROM Student;
   ```

3. 删除《数据库系统》课程的选课记录

   ```sql
   DELETE FROM SC
   WHERE cNo = (
     SELECT
       cNo
     FROM Course
     WHERE cName = '数据库系统'
   );
   ```

:::
