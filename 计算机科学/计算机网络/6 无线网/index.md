# 第 6 章 无线网

无线网络解决无线链路和移动性的问题，支持随时随地的网络接入。

## 6.1 无线链路基础

无线链路和有线的差异：

- **信号衰减**：信号传播时衰减。
- **干扰**：其他信号的干扰。
- **多径传播**：信号反射，导致多径时延。
- **隐藏站 / 暴露站问题**：节点无法侦听到所有的传输。
- **信噪比（`SINR`）**：信号和干扰噪声的比值，决定了传输速率和误码率。

### `CDMA`：码分多址

每个用户分配唯一的码片序列，用码片编码数据，接收方用码片解码，多个用户可以同时传输，互相不干扰，用于 3G/4G。

## 6.2 IEEE 802.11（`WiFi`）

`WiFi` 是无线局域网的标准，基于 `CSMA/CA`。

### 标准

- `802.11b`：2.4GHz，11Mbps。
- `802.11a/g`：5GHz/2.4GHz，54Mbps。
- `802.11n`：MIMO，200Mbps。

### 架构

- **`BSS`（基本服务集）**：接入点（AP，基站）+ 无线主机，AP 连接有线网络。
- **关联**：主机扫描信道，选择 AP 关联，获取 IP 地址。

### `CSMA/CA`：碰撞避免

无线无法做碰撞检测，因此采用碰撞避免：

- 先听后发，信道空闲 `DIFS` 时间后发送。
- 忙的话，随机退避，倒计时，到期发送。
- 接收方收到帧后，返回 `ACK`，因为无线的隐藏站问题，需要确认。
- `RTS/CTS`：发送方先发送 `RTS`，AP 回复 `CTS`，预留信道，避免长数据帧的冲突。

## 6.3 蜂窝网络与移动性

### 蜂窝网络发展

- **2G**：语音，GSM/CDMA。
- **3G**：语音 + 数据，WCDMA/CDMA2000。
- **4G/5G**：高速数据，LTE/5G。

### 移动性

移动用户切换基站时，通过间接路由实现：

- **本地代理**：`home` 网络的代理，拦截发给移动节点的分组。
- **外地代理**：`foreign` 网络的代理，转发分组给移动节点。
- **隧道**：将原始分组封装，转发给外地代理，解封装后发给移动节点。
