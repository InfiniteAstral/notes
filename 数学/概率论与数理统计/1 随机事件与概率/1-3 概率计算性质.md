# 1.3 概率计算性质

## 1.3.1 有界性

$$0 \leq P(A) \leq 1$$

## 1.3.2 规范性（归一性）

$$P(\emptyset) = 0$$

$$P(\omega) = 1$$

## 1.3.3 单调性

若 $A \subseteq B$，则 $P(A) \leq P(B)$

## 1.3.4 有限可加性

若事件 $A_1, A_2, \cdots, A_n$ 两两互斥，则

$$P\left(\bigcup_{i=1}^n A_i\right) = \sum_{i=1}^n P(A_i)$$

## 1.3.5 对立事件概率

$$P(\bar{A}) = 1 - P(A)$$

## 1.3.6 减法公式

$$P(A - B) = P(A\bar{B}) = P(A) - P(AB)$$

## 1.3.7 加法公式

$$P(A + B) = P(A) + P(B) - P(AB)$$

$$P(A + B + C) = P(A) + P(B) + P(C) - P(AB) - P(AC) - P(BC) + P(ABC)$$
