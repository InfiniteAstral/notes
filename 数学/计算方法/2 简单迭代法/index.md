# 简单迭代法

::: info 阶数
简单迭代法是**一阶迭代法**。
:::

对于**非线性方程** $f(x)=0$，若存在一个函数 $g(x)$，使得 $x=g(x)$，则称 $g(x)$ 是 $f(x)=0$ 的一个**迭代函数**，可以进行迭代计算。

这个过程**并不总是收敛，有可能会发散**。所以使用简单迭代法计算前要判断是否收敛：

## 全局收敛定理

::: warning 一般不用全局收敛定理
简单迭代法一般使用下面的 [局部收敛定理](#局部收敛定理)。
:::

若 $g(x)$ 在区间 $[a,b]$ 内，有：

1. **压缩性**：$g'(x)$ 连续且存在 $0<L<1$ 使得 $\left | g'(x) \right | \le L$
2. **映内性**：$\forall x \in [a,b]$ 使得 $g(x) \in [a,b]$

则：

1. **有唯一解**：$x=g(x)$ 在 $[a,b]$ 上有唯一定根 $x^*$
2. **任意初值都收敛**：迭代公式 $x_{k+1} = g(x_k)$ 对 $\forall x \in [a,b]$ 均收敛

## 局部收敛定理

若在 $x=g(x)$ 的根 $x^*$ 的某邻域内，有：

$g'(x)$ 连续，且存在 $0<L<1$ 使得 $\left | g'(x) \right | \le L$（**压缩性**），

则对这个邻域内的任意 $x$，简单迭代法均收敛。

::: tip 提示
要先通过零点存在性定理判断这个区间是否为隔根区间，再使用局部收敛定理。
:::
