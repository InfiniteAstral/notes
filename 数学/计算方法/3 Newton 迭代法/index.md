# Newton 迭代法

::: info 阶数
Newton 迭代法**至少**是**二阶迭代法**。
:::

## 迭代公式

$$x_{k+1}=x_k-\frac{f(x_k)}{f'(x_k)}$$

## 局部收敛定理

::: warning 一般不用局部收敛定理
Newton 迭代法一般使用下面的 [全局收敛定理](#全局收敛定理)。
:::

若在 $f(x)=0$ 的根 $x^*$ 的某邻域内，有 $f''(x)$ 连续，且 $f'(x^*)=0$，则对这个邻域内任意初值 $x_0$，Newton 迭代法均收敛。

## 全局收敛定理

若 $f(x)=0$ 的根 $x^*\in [a,b]$，且 $\forall x\in [a,b]$，且 $f'(x^*)\ne 0$，$f''(x^*)\ne 0$，则在 $[a,b]$ 内，能使 $f(x_0)f''(x_0)>0$ 的初值 $x_0$ 均能使 Newton 迭代法收敛。
